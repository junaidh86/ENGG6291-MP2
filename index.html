PHASE 1

Multi-Criteria Supplier Selection · AHP

Evaluate four global suppliers across four criteria using the Analytic Hierarchy Process. Each pairwise matrix is checked for consistency — any matrix with CR > 0.10 blocks the analysis until you revise it.

1.1 Criteria pairwise comparison

Use Guided judgments to say which criterion matters more and by how much (Saaty 1–9) — the system builds the comparison matrix for you. Switch to Matrix view to edit cells directly.

1.2 Supplier comparisons under each criterion

Compare the four suppliers under each criterion. Switch criteria with the tabs — the dot turns green once a matrix is consistent.

1.3 Final synthesis & ranking

Overall score = Σ (criterion weight × supplier weight under that criterion). The gate below stays locked until every matrix is consistent.

PHASE 3

Last-Mile Delivery · TSP

The winning supplier's goods are distributed from the port of entry across six regional hubs. Define a route using the distance matrix below, calculate the total route distance, and find the shortest closed tour. No sub-tour is allowed.
